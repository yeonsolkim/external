**Definition 1.4.1.** Let $\mathcal L_{\mathrm{field}}$ be the first-order language $\mathcal L_{\mathrm{field}} = \lbrace 0, 1, +, -, \cdot\rbrace,$ where $0$ and $1$ are constant symbols, $+$ and $\cdot$ are binary function symbols, and $-$ is a unary function symbol.<br>
<br>

**Definition 1.4.2.** Let $T_{\mathrm{field}}$ be the $\mathcal L_{\mathrm{field}}$-theory which consists of the following axioms:

1. $\forall a,b,c \ ((a + b) + c = a + (b + c)),$
2. $\forall a, b \ (a + b = b + a),$
3. $\forall a \ (a + 0 = a),$
4. $\forall a \ (a + (-a) = 0),$
5. $\forall a, b, c \ ((a \cdot b) \cdot c = a \cdot (b \cdot c)),$
6. $\forall a, b \ (a \cdot b = b \cdot a),$
7. $\forall a \ (a \cdot 1 = a),$
8. $\forall a, b, c \ (a \cdot (b + c) = (a \cdot b) + (a \cdot c)),$
9. $0 \neq 1,$
10. $\forall a \ (a \neq 0 \rightarrow \exists b \ (a \cdot b = 1)).$
{:marker-prefix="F" }
<br>

**Definition 1.4.3.** An $\mathcal L_{\mathrm{field}}$-structure ${\mathcal F}=(\mathbb F, 0, 1, +, -, \cdot)$ is called a *field* if ${\mathcal F}\vDash T_{\mathrm{field}},$ and $a$ is called a *scalar* if $a\in {\mathbb F}.$ $0$ and $1$ are called the *additive identity* and *multiplicative identity* respectively, and $-a$ is called the *additive inverse* of $a.$ <br>
