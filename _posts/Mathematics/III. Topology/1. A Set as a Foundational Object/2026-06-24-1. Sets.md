**Remark 1.1.1.** We take a set to be a foundational object and we shall redefine some mathematical objects based on sets. This is because, although we will be working on object-level, we should observe ZFC to avoid fatal logical errors, and also because the objects redefined in such way are indeed equivalent to the ordinary objects defined before.<br>
<br>

**Notation 1.1.2.** If an object $a$ belongs to set $A,$ we express this by writing  $a\in A.$ If $a$ does not belong to $A,$ we express this by writing $a\notin A.$ If two sets $A$ and $B$ consist of precisely the same elements, we denotes this fact as $A = B.$ If not, then we write $A\neq B.$ <br>
&emsp; If $A$ is a subset of $B,$ that is, if every element of $A$ is also an element of $B,$ we write $A\subset B.$ We also write $B\supset A,$ which is read "$B$ contains $A.$" If $A\subset B$ and $A\neq B,$ we write $A\subsetneq B,$ which is read "$A$ is a proper subset of $B.$"<br>
&emsp; If a set $A$ consists of the elements $a,b,$ and $c,$ we write $A=\lbrace a,b,c\rbrace.$ If a set $A$ consists of all elements of a set $B$ having some property $\varphi(x),$ we write $A=\lbrace x\in B : \varphi(x) \rbrace.$<br>
<br>

**Definition 1.1.3.** Let $A$ and $B$ be sets. We define the *union* of $A$ and $B$ as $\lbrace x : x\in A \lor x\in B\rbrace,$  denoted by $A\cup B.$ We also define the *intersection* of $A$ and $B$ as $\lbrace x : x\in A \land x\in B\rbrace,$ denoted by $A\cap B.$ The set consisting of no elements is called the *empty set,* denoted by $\varnothing.$ <br>
<br>

**Theorem 1.1.4.** Let $A,B,$ and $C$ be sets. Then following holds: 
1. $A\cap (B \cup C) = (A\cap B) \cup (A \cap C).$
2. $A\cup (B \cap C) = (A\cup B) \cap (A \cup C).$

<br>

**Definition 1.1.5** Let $A$ and $B$ be sets. The *relative complement* of $B$ in $A$ is the set $A-B$ defined by the equation $A-B=\lbrace x\in A:x\notin B\rbrace.$ <br>
<br>

**Theorem 1.1.6.** Let $A,B,$ and $C$ be sets. Then following holds: 

1. $A- (B \cup C) = (A- B) \cup (A - C).$
2. $A- (B \cap C) = (A- B) \cap (A - C).$

<br>

**Definition 1.1.7.** The *power set* of a set $A$ is the collection $\mathcal P$ defined by the equation $\mathcal P=\lbrace x : x\subset A  \rbrace.$ We also denote the set as $\mathcal P(A).$<br>
<br>

**Definition 1.1.8.** We consider the union and intersection of arbitrary many sets. We first form the collection $\mathcal C$ of sets. Then we define the union of the sets in $\mathcal C$ as $\lbrace x : \exists y \in \mathcal C \ ( x\in y) \rbrace,$ denoted by $\bigcup \mathcal C.$ Also we define the intersection of the sets in $\mathcal C$ as $\lbrace x : \forall y \in \mathcal C \ (x\in y) \rbrace,$ denoted by $\bigcap \mathcal C.$<br>
<br>

**Definition 1.1.9.** We now consider a set of ordered pairs, which are also called 2-tuples. Given sets $A$ and $B,$ we define their *Cartesian product* $A\times B$ to be the set of all ordered pairs $(a,b)$ where $a\in A$ and $b\in B.$ That is, $A\times B = \lbrace (a,b) : a\in A \land b \in B \rbrace.$
