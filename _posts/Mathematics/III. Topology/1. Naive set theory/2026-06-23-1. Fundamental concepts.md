**Notation 1.1.1.** *Membership* is a binary relation, denoted by $\in.$ For two sets $a$ and $A,$ 

$$a\in A$$

is read "$a$ is an element of $A$" or "$a$ belongs to $A.$" <br>
&emsp; We say that $A$ is a *subset* of $B$ if 

$$\forall x\,(x\in A \Rightarrow x\in B).$$ 

We simply express this by writing $A\subset B.$ 

<br>

**Principle 1.1.2.** Two sets are equal exactly when they have the same elements, that is,

$$\forall A\, \forall B\ [\forall x\ (x\in A\Leftrightarrow x\in B) \Rightarrow A=B]. $$

<br> 

**Principle 1.1.3.**  There is a set which has no elements, that is, 

 $$\exists E\, \forall x\ (x \notin E).$$

The usual symbol for that set, which is called the *empty set,* is 

$$ \varnothing.$$

<br>


**Principle 1.1.4.** We can create a new set by carving out a set which already exists. In other words,

$$\forall A\,\forall p\,\exists B \,\forall x\ [x\in B\Leftrightarrow x\in A\wedge \varphi (x,p)].$$

The set newly created is denoted by $\lbrace x\in A:\varphi(x,p)\rbrace.$ <br>
<br>

**Principle 1.1.5.** There exists the set which consists exactly of two sets $a$ and $b,$ that is, 

$$\forall a\,\forall b\,\exists B\,\forall x\ [x\in B\Leftrightarrow \varphi(x,a,b)] \quad  \text{ where }\varphi(x,a,b):x=a\lor x=b.$$

The newly created is denoted by $\lbrace a,b\rbrace,$ and called the *unordered pair* of $a$ and $b.$ We may refer to this principle as a pseudo-special case of the principle of specification in the sense that if there were a universe, then the principle of paring would follow as a special case. <br>
&emsp; Since we plan to introduce such generating principles, it will be useful to introduce a new notation. If $\varphi(x,p)$ is a condition such that the sets that $\varphi(x,p)$ specifies constitute a set, then we denote that set as $\lbrace x:\varphi(x,p)\rbrace.$<br>
<br>

**Principle 1.1.6.** There exists a comprehensive set, that is, 

$$\forall A\,\exists B\,\forall x\ [ \exists y\ (y\in A\wedge x\in y)\Rightarrow x\in B].$$

Applying the principle of specification, we have

$$\forall A\,\exists B\,\forall x\ [ \exists y\ (y\in A\wedge x\in y)\Leftrightarrow x\in B].$$

The set newly created is denoted by $\bigcup A$ and we call the set the *union* of $A.$ In particular, we use a  special notation for the union of a pair: 

$$X\cup Y=\bigcup\lbrace X,Y\rbrace.$$ 

Since $X\cup Y=\lbrace x:x\in X\lor x\in Y\rbrace,$ we can generalize pairs: $\lbrace a,b,c\rbrace =\lbrace a\rbrace \cup \lbrace b\rbrace \cup \lbrace c\rbrace , \text{etc.}$<br>
<br>

**Definition 1.1.7.** *Relative complement* of $B$ in $A$ is the set $A-B$ defined by 

$$A-B=\lbrace x\in A:x\notin B\rbrace.$$

Dealing with sets which are subsets of $E,$ we can define *absolute complement*. Often used symbol for absolute complement of $A$ is $A'.$ <br>
<br>

**Theorem 1.1.8.** There are two important statements about unions and intersections, which is so-called *De Morgan laws:*

1. $$\forall A\,\forall B\ [(A\cup B)'=A'\cap B'].$$
2. $$\forall A\,\forall B\ [(A\cap B)'=A'\cup B'].$$

<br>

**Principle 1.1.9.** For each set there exists a collection of sets that contains among its elements all the subsets of the given set, that is,

$$\forall E\, \exists \mathcal{P}\, \forall x\ (x\subset E\Rightarrow x\in \mathcal{P}).$$ 

Applying axiom of specification, we have 

$$\forall E\, \exists \mathcal{P}\, \forall x\ (x\subset E\Leftrightarrow x\in \mathcal{P}).$$

The set $\mathcal{P}$ is called the *power set* of $E;$ the dependence of $\mathcal{P}$ on $E$ is denoted by $\mathcal{P}(E).$
