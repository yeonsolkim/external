**Definition 1.1.1.** Let $\mathcal L_{\mathrm{field}}$ be the first-order language $\mathcal L_{\mathrm{field}} = \lbrace 0, 1, +, -, \cdot\rbrace,$ where $0$ and $1$ are constant symbols, $+$ and $\cdot$ are binary function symbols, and $-$ is a unary function symbol.<br>
<br>

**Definition 1.1.2.** Let $T_{\mathrm{field}}$ be the $\mathcal L_{\mathrm{field}}$-theory which consists of the following axioms:

1. $\forall x,y,z \ ((x + y) + z = x + (y + z)),$
2. $\forall x,y \ (x + y = y + x),$
3. $\forall x \ (x + 0 = x),$
4. $\forall x \ (x + (-x) = 0),$
5. $\forall x,y,z \ ((x \cdot y) \cdot z = x \cdot (y \cdot z)),$
6. $\forall x,y \ (x \cdot y = y \cdot x),$
7. $\forall x \ (x \cdot 1 = x),$
8. $\forall x,y,z \ (x \cdot (y + z) = (x \cdot y) + (x \cdot z)),$
9. $0 \neq 1,$
10. $\forall x \ (x \neq 0 \rightarrow \exists y \ (x \cdot y = 1)).$
{:marker-prefix="F" }

<br>

**Definition 1.1.3.** An $\mathcal L_{\mathrm{field}}$-structure ${\mathcal F}=(\mathbb F, 0, 1, +, -, \cdot)$ is called a *field* if ${\mathcal F}\models T_{\mathrm{field}},$ and $a$ is called a *scalar* if $a\in {\mathbb F}.$ $0$ and $1$ are called the *additive identity* and *multiplicative identity* respectively, and $-a$ is called the *additive inverse* of $a.$ <br>
<br>

**Definition 1.1.4.** Let $\mathcal{L}_{\mathrm{of}} = \mathcal{L}_{\mathrm{field}} \cup \{<\},$ where $<$ is a binary relation symbol, and let $T_{\mathrm{of}}$ be the $\mathcal{L}_{\mathrm{of}}$-theory that consists of the field axioms together with the following five axioms: 

1. $\forall x \ \neg(x < x),$
2. $\forall x,y,z\ ((x < y \land y < z) \rightarrow x < z),$
3. $\forall x,y\ (x < y \lor x = y \lor y < x),$
4. $\forall x,y,z\ (x < y \rightarrow x + z < y + z),$
5. $\forall x,y\ ((0 < x \land 0 < y) \rightarrow 0 < x \cdot y).$
{:marker-prefix="OF" }

An $\mathcal{L}_{\mathrm{of}}$-structure $\mathcal{F} = (\mathbb{F}, 0, 1, +, -, \cdot, <)$ is called an ordered field if $\mathcal{F} \models T_{\mathrm{of}}.$<br>
<br>

**Theorem 1.1.5.** The real number system satisfies $T_{\mathrm{of}},$ that is,

$$
\mathcal R=(\mathbb R,0,1,+,-,\cdot,<) \models T_{\mathrm{of}}.
$$

And it has the least-upper-bound property. Furthermore, $\mathbb Q$ is an ordered subfield of $\mathbb R.$<br>
<br>

**Corollary 1.1.6** (Archimedean property). If $x,y\in \mathbb R$ and $0<x,$ then there is a positive integer $n$ such that 

$$ y < nx.$$

<br>

## Exercises.

**Exercise 1.** Prove that if $x$ is positive then the negative of $x$ is negative.<br><br>
*Proof.* We recall Theorem 1.1.3: If there is a relation between $x$ and $y,$ then for any number $z,$ there is a relation between the sum of $x$ and $z,$ and the sum of $y$ and $z.$ Returning to the problem, we have the relation between the $0$ and $x,$ $0<x.$ We also have another binary relation such that $x+(-x)=0.$ Then we can compare $x+(-x)$ and $x,$ that is, $x+(-x)<x.$ Thus $(-x) + (x+(-x)) < x+(-x),$ so that $-x<0.$ Therefore the negative of $x$ is negative. <span class="qed">$\square$</span> <br>
<br>

**Exercise 2.** Prove that the product of two negative numbers $x$ and $y,$ namely $xy$ is positive.<br><br>
*Proof.* We have $0<x.$ Since $0<y,$ it follows that $0y<xy.$ But we also have the relation between $0$ and $0y,$ $0=0y.$ Therefore $0<xy,$ and $xy$ is positive.<span class="qed">$\square$</span><br>
<br>

**Exercise 3.** Prove that if $x<y$ and $z<w$ then $x+z<y+w.$<br><br>
*Proof.* Since $x<y,$ it follows that $x+z<y+z$ by Theorem 1.1.3. Similarly, since $z<w,$ we have  $z+y<w+y.$ Since the order relation is transitive, $x+z<w+y.$<span class="qed">$\square$</span><br>
<br>

**Exercise 4.** Let $x\in \mathbb R.$ Prove that $0\le x^2.$<br><br>
*Proof.* Since the order relation is comparable if $0\ne x$ then $0<x$ or $x<0.$ If $0<x$ then $0x<xx,$ so that $0<x^2.$ If $x<0$ then $0<x^2$ holds from the consequence of Exercise 2. The only case left is that $x=0.$ If $x=0$ then $x^2=xx=0.$ Therefore $0\le x^2.$ <span class="qed">$\square$</span><br>
