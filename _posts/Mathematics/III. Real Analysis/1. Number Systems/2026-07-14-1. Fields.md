**Definition 1.1.1.** Let $\mathcal L_{\mathrm{field}}$ be the first-order language $\mathcal L_{\mathrm{field}} = \lbrace 0, 1, +, -, \cdot\rbrace,$ where $0$ and $1$ are constant symbols, $+$ and $\cdot$ are binary function symbols, and $-$ is a unary function symbol.<br>
<br>

**Definition 1.1.2.** Let $T_{\mathrm{field}}$ be the $\mathcal L_{\mathrm{field}}$-theory which consists of the following axioms:

1. $\forall x,y,z \ ((x + y) + z = x + (y + z)),$
2. $\forall x,y \ (x + y = y + x),$
3. $\forall x \ (x + 0 = x),$
4. $\forall x \ (x + (-x) = 0),$
5. $\forall x,y,z \ ((x \cdot y) \cdot z = x \cdot (y \cdot z)),$
6. $\forall x,y \ (x \cdot y = y \cdot x),$
7. $\forall x \ (x \cdot 1 = x),$
8. $\forall x,y,z \ (x \cdot (y + z) = (x \cdot y) + (x \cdot z)),$
9. $0 \neq 1,$
10. $\forall x \ (x \neq 0 \rightarrow \exists y \ (x \cdot y = 1)).$
{:marker-prefix="F" }

<br>

**Theorem 1.1.3.** For all $x,y,z\in \mathbb R,$ 
1. $x+y=x+z\Rightarrow y=z,$
2. $x+y=x\Rightarrow y=0,$ 
3. $x+y=0\Rightarrow y=-x,$
4. $-(-x)=x,$

*Proof.*<br>
(1) Since $x+y=x+z,$ $(x+y)+(-x) = (x+z) + (-x).$ Hence $y=z.$<br>
(2) Since $x+0=x,$ we have $x+y=x+0.$ Hence $y=0,$ by (1).<br>
(3) Since $x+(-x)=0,$ we have $x+y=x+(-x).$ Hence $y=-x,$ by (1).<br>
(4) Since $x+(-x)=0,$ $-(-x)=x,$ by (3).<span class="qed">$\square$</span><br>
<br>

**Theorem 1.1.4.** For all $x,y,z\in \mathbb R,$ 
1. $(x\ne 0 \land x\cdot y=x\cdot z)\Rightarrow y=z,$
2. $(x\ne 0 \land x\cdot y=x)\Rightarrow y=1,$
3. $x\cdot y=(-x)\cdot (-y),$
4. $(-1)\cdot x = -x.$

*Proof.*<br>
(1) Let $x\cdot w=1.$ Since $(x\cdot y)\cdot w = (x\cdot z)\cdot w,$ $(x\cdot w)\cdot y = (x\cdot w)\cdot z.$ Hence $y=z.$<br>
(2) Since $x\cdot 1 = x,$ $x\cdot y = x\cdot 1.$ Since $x\ne 0,$ by (1), $y=1.$<br>
(3) We have $x\cdot y+(-x)\cdot y=0.$ Then, $(-x)\cdot y+(-x)\cdot (-y)=0.$ Since the additive inverse of $(-x)\cdot y$ is unique, $x\cdot y=(-x)\cdot (-y).$<br>
(4) Since $x+((-1)\cdot x)=(1+(-1))\cdot x = 0,$ by Theorem 1.1.3, $(-1)\cdot x = -x.$<span class="qed">$\square$</span><br>
<br>

**Definition 1.1.5.** $0$ and $1$ are called the *additive identity* and *multiplicative identity*, respectively. $-x$ is called the *additive inverse* of $x$ or the *negative* of $x.$ <br>
&emsp; We write $y=1/x,$ if $x\ne 0$ and $x\cdot y = 1.$ Further, we often write 

$$ x-y,xy,\frac x y,2x,3x,x^2,x^3,\dots $$ 

in place of 

$$ x+(-y),x\cdot y,x\cdot \left(\frac 1 y \right),x+x,x+x+x,x\cdot x,x\cdot x\cdot x,\dots .$$

&emsp; An $\mathcal L_{\mathrm{field}}$-structure ${\mathcal F}=(\mathbb F, 0, 1, +, -, \cdot)$ is called a *field* if ${\mathcal F}\vDash T_{\mathrm{field}},$ and $x$ is called a *scalar* if $x\in {\mathbb F}.$<br>
<br>

**Definition 1.1.6.** Let $\mathcal{L}_{\mathrm{of}} = \mathcal{L}_{\mathrm{field}} \cup \{<\},$ where $<$ is a binary relation symbol, and let $T_{\mathrm{of}}$ be the $\mathcal{L}_{\mathrm{of}}$-theory that consists of the field axioms together with the following five axioms: 

1. $\forall x \ \neg(x < x),$
2. $\forall x,y,z\ ((x < y \land y < z) \rightarrow x < z),$
3. $\forall x,y\ (x\ne y \rightarrow (x < y \lor y < x)),$
4. $\forall x,y,z\ (x < y \rightarrow x + z < y + z),$
5. $\forall x,y,z\ ((x < y \land 0 < z) \rightarrow x\cdot z < y \cdot z).$
{:marker-prefix="OF" start="11"}

An $\mathcal{L}_{\mathrm{of}}$-structure $\mathcal{F} = (\mathbb{F}, 0, 1, +, -, \cdot, <)$ is called an ordered field if $\mathcal{F} \vDash T_{\mathrm{of}}.$<br>
<br>

**Theorem 1.1.7.** For all $w,x,y,z\in \mathbb R,$ 
1. $0<x \Rightarrow -x<0,$
2. $(x<0\land y<0) \Rightarrow 0<xy,$
3. $(x<y\land z<w) \Rightarrow x+z<y+w,$
4. $0\le x^2.$

*Proof.*<br>
(1) Since $x+(-x)=0,$ we have $x+(-x)<x.$ Then $(x+(-x))+(-x)<x+(-x)$ by (OF14). Hence $-x<0.$<br>
(2) We have $0<x.$ Since $0<y,$ it follows that $0y<xy.$ But we also have the relation between $0$ and $0y,$ $0=0y.$ Therefore $0<xy.$<br>
(3) Since $x<y,$ it follows that $x+z<y+z.$ Similarly, since $z<w,$ we have  $z+y<w+y.$ Since the order relation is transitive, $x+z<w+y.$<br>
(4) Since the order relation is comparable if $0\ne x$ then $0<x$ or $x<0.$ If $0<x$ then $0x<xx,$ so that $0<x^2.$ If $x<0$ then $0<x^2.$ The only case left is that $x=0.$ If $x=0$ then $x^2=0x=0.$ Therefore $0\le x^2.$ <span class="qed">$\square$</span><br>
<br>

**Theorem 1.1.8.** The real number system $\mathcal R=(\mathbb R,0,1,+,-,\cdot,<)$ has the least-upper-bound property, and

$$
\mathcal R \vDash T_{\mathrm{of}}.
$$

<br>

**Theorem 1.1.9.** For all $x\in \mathbb R$ and $y\in \mathbb R_{>0},$ there exists $n\in \mathbb N$ such that 

$$x<ny.$$

*Proof.* Let $a\in \mathbb R$ and $b\in\mathbb R_{>0}$ be given, and let $A$ be the set of all $nb,$ where $n$ runs through the positive integers. We suppose, for contradiction, that $A$ is bounded above. Then $A$ has the supremum in $\mathbb R,$ by Theorem 1.1.5. Put $\alpha = \sup A.$ There is an element $mb$ such that $\alpha - b < mb.$ But then we have $\alpha < (m+1)b\in A,$ which contradicts that $\alpha$ is an upper bound for $A.$ Therefore it is concluded that the Theorem 1.1.6 is in fact true. <span class="qed">$\square$</span><br>
<br>

**Theorem 1.1.10.** For all $x,y\in \mathbb R,$ if $x<y$ then there exists $p\in \mathbb Q^{\ast}$ such that 

$$x<p<y.$$

*Proof.* There exists a positive integer $n$ such that $1<n(y-x),$ by Theorem 1.1.6. Applying Theorem 1.1.6 again, we obtain positive integers $m_1$ and $m_2$ such that $nx<m_1$ and $-nx<m_2.$ Then $-m_2<nx<m_1.$<br>

## Exercises.

**Exercise 1.** Prove that for all $x,y\in \mathbb R,$ 

1. $0<1,$
2. $x<y \Rightarrow -y<-x,$
3. $(0<x \land 0<xy) \Rightarrow 0<y,$
4. $0<x<y \Rightarrow 1/y<1/x,$
5. $x<y \Rightarrow x<(x+y)/2<y.$

*Proof.*<br>
(1) We have $0\le 1\cdot 1.$ Since $1\cdot 1 = 1$ and $0\ne 1,$ $0<1.$<br>
(2) Suppose, for contradiction, that $-x\le -y.$ Since $x<y,$ it follows that $x+(-x)<y+(-y),$ which contradicts $x+(-x)=y+(-y).$ Hence $-y<-x.$<br>
(3) Suppose, for contradiction, that $y\le 0.$ If $y=0,$ then $x\cdot y = 0,$ which contradicts $0<xy.$ If $y<0,$ then $0<-y,$ by (1). Thus $0<x(-y).$ It yields $0<-(xy),$ since $x(-y)= -(xy).$ Therefore $xy<0,$ which contradicts $0<xy.$ Consequently, it is concluded that $0<y.$<br>
(4) If $1/x\le 1/y$ then $x(1/x)<y(1/x)\le y(1/y).$ If follows that $x(1/x)<y(1/y),$ which contradicts $x(1/x)=1=y(1/y).$ Therefore it is concluded that $1/y< 1/x.$<span class="qed">$\square$</span><br>
