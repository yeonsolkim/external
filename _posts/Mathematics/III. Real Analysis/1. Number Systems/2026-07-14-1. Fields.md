**Definition 1.1.1.** Let $\mathcal L_{\mathrm{field}}$ be the first-order language $\mathcal L_{\mathrm{field}} = \lbrace 0, 1, +, -, \cdot\rbrace,$ where $0$ and $1$ are constant symbols, $+$ and $\cdot$ are binary function symbols, and $-$ is a unary function symbol.<br>
<br>

**Definition 1.1.2.** Let $T_{\mathrm{field}}$ be the $\mathcal L_{\mathrm{field}}$-theory which consists of the following axioms:

1. $\forall x,y,z \ ((x + y) + z = x + (y + z)),$
2. $\forall x,y \ (x + y = y + x),$
3. $\forall x \ (x + 0 = x),$
4. $\forall x \ (x + (-x) = 0),$
5. $\forall x,y,z \ ((x \cdot y) \cdot z = x \cdot (y \cdot z)),$
6. $\forall x,y \ (x \cdot y = y \cdot x),$
7. $\forall x \ (x \cdot 1 = x),$
8. $\forall x,y,z \ (x \cdot (y + z) = (x \cdot y) + (x \cdot z)),$
9. $0 \neq 1,$
10. $\forall x \ (x \neq 0 \rightarrow \exists y \ (x \cdot y = 1)).$
{:marker-prefix="F" }

<br>

**Definition 1.1.3.** An $\mathcal L_{\mathrm{field}}$-structure ${\mathcal F}=(\mathbb F, 0, 1, +, -, \cdot)$ is called a *field* if ${\mathcal F}\vDash T_{\mathrm{field}},$ and $a$ is called a *scalar* if $a\in {\mathbb F}.$ $0$ and $1$ are called the *additive identity* and *multiplicative identity* respectively, and $-a$ is called the *additive inverse* of $a.$ <br>
<br>

**Definition 1.1.4.** Let $\mathcal{L}_{\mathrm{of}} = \mathcal{L}_{\mathrm{field}} \cup \{<\},$ where $<$ is a binary relation symbol, and let $T_{\mathrm{of}}$ be the $\mathcal{L}_{\mathrm{of}}$-theory that consists of the field axioms together with the following five axioms: 

1. $\forall x \ \neg(x < x),$
2. $\forall x,y,z\ ((x < y \land y < z) \rightarrow x < z),$
3. $\forall x,y\ (x\ne y \rightarrow (x < y \lor y < x)),$
4. $\forall x,y,z\ (x < y \rightarrow x + z < y + z),$
5. $\forall x,y,z\ ((x < y \land 0 < z) \rightarrow x\cdot z < y \cdot z).$
{:marker-prefix="OF" start="11"}

An $\mathcal{L}_{\mathrm{of}}$-structure $\mathcal{F} = (\mathbb{F}, 0, 1, +, -, \cdot, <)$ is called an ordered field if $\mathcal{F} \vDash T_{\mathrm{of}}.$<br>
<br>

**Theorem 1.1.5.** The real number system $\mathcal R=(\mathbb R,0,1,+,-,\cdot,<)$ has the least-upper-bound property, and

$$
\mathcal R \vDash T_{\mathrm{of}}.
$$

<br>

**Theorem 1.1.6.** For all $x\in \mathbb R$ and $y\in \mathbb R_{>0},$ there exists $n\in \mathbb N$ such that 

$$x<ny.$$

*Proof.* Let $a\in \mathbb R$ and $b\in\mathbb R_{>0}$ be given, and let $A$ be the set of all $nb,$ where $n$ runs through the positive integers. We suppose, for contradiction, that $A$ is bounded above. Then $A$ has the supremum in $\mathbb R,$ by Theorem 1.1.5. Put $\alpha = \sup A.$ There is an element $mb$ such that $\alpha - b < mb.$ But then we have $\alpha < (m+1)b\in A,$ which contradicts that $\alpha$ is an upper bound for $A.$ Therefore it is concluded that the Theorem 1.1.6 is in fact true. <span class="qed">$\square$</span><br>
<br>

**Theorem 1.1.7.** For all $x,y\in \mathbb R,$ if $x<y$ then there exists $p\in \mathbb Q^{\ast}$ such that 

$$x<p<y.$$

*Proof.* There exists a positive integer $n$ such that $1<n(y-x),$ by Theorem 1.1.6. Applying Theorem 1.1.6 again, we obtain positive integers $m_1$ and $m_2$ such that $nx<m_1$ and $-nx<m_2.$ Then $-m_2<nx<m_1.$

## Exercises.

**Exercise 1.** Let $w,x,y,z\in \mathbb R.$ Prove that
1. $0<x \Rightarrow -x<0,$
2. $(x<0\land y<0) \Rightarrow 0<xy,$
3. $(x<y\land z<w) \Rightarrow x+z<y+w,$
4. $0\le x^2,$
5. $0<1,$
6. $x<y \Rightarrow -y<-x,$
7. $0<x<y \Rightarrow 1/y<1/x,$
8. $x<y \Rightarrow x<(x+y)/2<y.$

*Proof.* 
(1) We recall (OF14): If there is a relation between $x$ and $y,$ then for any number $z,$ there is a relation between the sum of $x$ and $z,$ and the sum of $y$ and $z.$ Returning to the problem, we have the relation between the $0$ and $x,$ $0<x.$ We also have another binary relation such that $x+(-x)=0.$ Then we can compare $x+(-x)$ and $x$; we have $x+(-x)<x.$ Thus $(-x) + (x+(-x)) < x+(-x),$ so that $-x<0.$ Therefore the negative of $x$ is negative.<br><br>
(2) We have $0<x.$ Since $0<y,$ it follows that $0y<xy.$ But we also have the relation between $0$ and $0y,$ $0=0y.$ Therefore $0<xy,$ and $xy$ is positive.<br><br>
(3) Since $x<y,$ it follows that $x+z<y+z.$ Similarly, since $z<w,$ we have  $z+y<w+y.$ Since the order relation is transitive, $x+z<w+y.$<br><br>
(4) Since the order relation is comparable if $0\ne x$ then $0<x$ or $x<0.$ If $0<x$ then $0x<xx,$ so that $0<x^2.$ If $x<0$ then $0<x^2.$ The only case left is that $x=0.$ If $x=0$ then $x^2=0x=0.$ Therefore $0\le x^2.$ <br><br>
(5) We have $0\le 1\cdot 1.$ Since $1\cdot 1 = 1$ and $0\ne 1,$ $0<1.$<br><br>
(6) Suppose, for contradiction, that $-x\le -y.$ It yields $x+(-x)<y+(-y),$ since $x<y,$ which contradicts $x+(-x)=y+(-y).$ Hence $-y<-x.$
