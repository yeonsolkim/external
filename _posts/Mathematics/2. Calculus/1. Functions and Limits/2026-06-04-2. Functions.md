**Remark 1.2.1.** We recall the definition of a function. Let $A$ and $B$ be sets. A rule of assignment $R$ is a subset of the product $A\times B$ such that 

$$ (a,b)\in R \land (a,b')\in R  \Longrightarrow b=b'.$$

A rule of assignment $R$ whose domain is $A,$ together with a set $B$ that contains the image of $R$ is a function from $A$ to $B.$ When we denote the function as $f,$ we write 

$$f: A\longrightarrow B.$$

<br>

**Definition 1.2.2.** Let $f:D\to \mathbb R$ be a function and let $I\subseteq D.$ $f$ *increases* on $I$ if 

$$ \forall x_1,x_2\in I\ (x_1<x_2 \Rightarrow f(x_1)\le f(x_2)).$$

Conversely, $f$ *decreases* on $I$ if 

$$ \forall x_1,x_2\in I\ (x_1<x_2 \Rightarrow f(x_2)\le f(x_1)).$$

<br>

**Definition 1.2.3.** A function $f:\mathbb R \to \mathbb R$ is a *polynomial function* if there exist numbers $n\in \mathbb N$ and $c_n,\dots, c_0\in \mathbb R$ such that for every $x\in \mathbb R$

$$f(x)=c_nx^n+c_{n-1}x^{n-1}+\cdots+c_1x+c_0.$$

In particular if 

$$f(x)=c_2x^2+ c_1x+ c_0$$

for every $x\in \mathbb R$ and $c_2\ne 0,$ then the function is called a *quadratic function.*<br>
<br>

**Theorem 1.2.4.** Let $f:\mathbb R \to \mathbb R$ be a quadratic function such that 

$$ \forall x\in \mathbb R\ (f(x)=c_2x^2+c_1x+c_0),$$

and let $\Delta = c_1^2-4c_2c_0.$ The equation $f(x)=0$ has no real roots if and only if  

$$\Delta< 0.$$

Also, the equation has one real roots if and only if

$$ \Delta = 0,$$

and has two distinct real roots if and only if 

$$ 0<\Delta.$$

<br>

**Definition 1.2.5.** A function $f:D\to\mathbb{R}$ is a *rational function* if there exist polynomial functions $p,q:\mathbb{R}\to\mathbb{R},$ with $q$ not the zero polynomial function such that

$$ f(x)=\frac{p(x)}{q(x)} $$

for every $x\in D,$ where $D=\lbrace x\in\mathbb{R}:q(x)\neq 0\rbrace.$<br>
<br>

**Definition 1.2.6.** A function $f:D\to\mathbb{R}$ is a *root function* if there exists a positive integer $n$ such that

$$

f(x)=\sqrt[n]{x}

$$

for every $x\in D,$ where $D=[0,\infty)$ if $n$ is even and $D=\mathbb{R}$ if $n$ is odd.<br>
<br>

**Definition 1.2.7.** A function $f$ is an *algebraic function* if it can be constructed using algebraic operations—such as addition, subtraction, multiplication, division, and taking roots—starting with polynomials. And, a function $f$ is a *transcendental function* if it can't be constructed using the algebraic operations.<br>
<br>

**Definition 1.2.8** (Trigonometric functions). For each $x\in \mathbb R,$ let $P_x=(a(x),b(x))$ be the point on the unit circle obtained by moving a signed arc length $x$ from the point $(1,0),$ counterclockwise if $x>0$ and clockwise if $x<0.$ The *sine function* and the *cosine function* are the functions $\sin,\cos:\mathbb R\to\mathbb R$ such that

$$\sin x=b(x),$$

and

$$\cos x=a(x)$$

for every $x\in\mathbb R.$ The *tangent function* is the function $\tan:D\to\mathbb R$ such that

$$\tan x=\frac{\sin x}{\cos x}$$

for every $x\in D,$ where $D=\lbrace x\in\mathbb R:\cos x\neq 0\rbrace .$
